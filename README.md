## Ch3_4 Vue인스턴스

- vue생성 방식.
    - 1. el속성을 이용.
    ```js
    new Vue(){
        el : ~
    }
    ```
    - 2. 인스턴스 마운트 메서드를 사용하여 생성(new Vue~.$mouunt) 이 경우는 UI 조작이나 통신 등의 작업을 진행한 후에 화면 요소가 생성되면 $mount() 뷰 인스턴스 메서드를 이용하여 마운트
    ```js
    new Vue({

    }).$mount('#app')
    ```
    - 3. vm 변수를 선언하여 생성 -> 이 경우 뷰 인스턴스 생성 후 뷰 인스턴스를 활용하고자 할 때 사용.
    ```js
    var vm = new Vue({

    })
    vm.$mount('#app')
    ```
